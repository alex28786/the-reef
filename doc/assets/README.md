## Usage

These assets can be used for:
- Documentation and README files
- Issue reports and bug tracking
- Stakeholder presentations
- Onboarding new developers

## Adding New Assets

When adding new screenshots or recordings:
1. Use descriptive filenames (feature_action.ext)
2. Add entry to this README
3. Reference in appropriate documentation
